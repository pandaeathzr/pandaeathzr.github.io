关于parseInt

parseInt(string, radix) 函数解析一个字符串参数，并返回一个指定基数的整数 当radix = undefined || 0 时,采用默认的行为：

  • 如果字符串 string 以"0x"或者"0X"开头, 则基数是16 (16进制).
  • 如果字符串 string 以"0"开头, 基数是8（八进制）或者10（十进制），那么具体是哪个基数由实现环境决定。Node使用的基数是10。因此，永远都要明确给出radix参数的值。
  • 如果字符串 string 以其它任何值开头，则基数是10 (十进制)。

当 radix = 1 时，似乎输出的都是 NaN。

当 传进来的参数 不是radix进制时，返回的也是 NaN
