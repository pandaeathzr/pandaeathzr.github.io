我不知道的深浅拷贝

2019-09-20

什么是深拷贝，什么是浅拷贝在这里就不谈了，因为我已经知道了！

浅拷贝的实现方式：

  • Object.assign()
  • 展开符运算 ...
  • Array.prototype.slice()

Object.assign(target, ...sources)

方法用于将所有可枚举属性的值从一个或多个源对象复制到目标对象。它将返回目标对象

继承属性和不可枚举属性是不能拷贝的

原始类型会被包装为对象

这是在原始类型有默认迭代器的情况下，例如 String

深拷贝的实现方式：

  • JSON.parse(JSON.stringify(Object))
  • JQuery.extend()
  • lodash.cloneDeep

JSON.parse(JSON.stringify(Object))缺陷

  • 会忽略 undefined
  • 会忽略 symbol
  • 不能序列化函数
  • 不能解决循环引用的对象
  • 不能处理正则

博文中介绍说处理Date()时会转换错误，但是经过我的测试是不会的。
